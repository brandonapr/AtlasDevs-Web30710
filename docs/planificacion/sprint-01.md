# Sprint 01

## Objetivo
Dejar listo el sistema de trabajo colaborativo del proyecto y cerrar la planificacion inicial del MVP.

## Items comprometidos
- EDV-01 Estrategia de ramas y reglas Git.
- EDV-02 Tablero Jira y backlog inicial.
- EDV-03 Capacitacion base del equipo en Git y Jira.
- EDV-04 Inventario fotografico de platos.
- EDV-05 Mockups iniciales del flujo principal.
- EDV-06 Validacion del flujo de caja con cliente.

## Criterio de cierre
- Existe backlog visible y priorizado.
- Cada integrante tiene tareas asignadas.
- Git y Jira ya reflejan el flujo real de trabajo.
- El equipo puede abrir requests, ramas y pull requests con una convension comun.
